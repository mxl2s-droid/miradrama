# Mira Drama Developer Site

Static Google Play developer website for Sabir and Mira Drama short-drama recommendation apps.

## Included files

- `index.html` - English homepage
- `index-zh.html` - Chinese homepage
- `privacy.html` - privacy policy for Google Play and in-app links
- `privacypolicy/index.html` - compatibility redirect path
- `support.html` - support contact page
- `terms.html` - app terms page
- `app-ads.txt` - mobile app authorized sellers file
- `ads.txt` - optional web ads seller file
- `robots.txt` and `sitemap.xml` - search crawler files
- `CNAME` - GitHub Pages custom domain for `mira-drama.com`
- `404.html` - GitHub Pages fallback page
- `assets/icon-192.png` - app logo and site icon
- `assets/mira-drama-recommendation-hero.png` - generated recommendation-list hero image
- `assets/site.webmanifest` - web app manifest
- `google-play-listing.md` - copy-ready Play Console fields

## Before publishing

1. Replace `pub-0000000000000000` in `app-ads.txt` with the real AdMob publisher ID.
2. Push this folder to a GitHub repository.
3. Enable GitHub Pages from the repository root or `/docs` depending on your repo setup.
4. Point the DNS records for `mira-drama.com` to GitHub Pages.
5. In Google Play Console, set the developer website to `https://mira-drama.com/`.
6. Use `https://mira-drama.com/privacy.html` as the privacy policy URL.
